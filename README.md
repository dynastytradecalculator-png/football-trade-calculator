# 🏈 Dynasty Football Trade Calculator

A simple fantasy football trade analyzer tool.

## 🔥 Features
- Compare Team A vs Team B trade value
- Instant result (Win / Loss / Fair)
- Lightweight HTML, CSS, JavaScript
- Works on GitHub Pages

## 🚀 How to Use
1. Open index.html in browser
2. Enter team/player values
3. Click "Analyze Trade"

## 🌐 Live Demo
Deploy using GitHub Pages:
Settings → Pages → Deploy from branch
